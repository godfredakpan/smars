  • blog-without-sidebar.html is for Blog List Style Page (Without Sidebar)

  • blog-grid-without-sidebar.html is for Blog Grid Style Page (Without Sidebar)

These are the css files that are loaded into templates in Head Section.

            <link rel="stylesheet" href="assets/css/slick.css">
